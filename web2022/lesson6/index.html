JavaScript

JavaScript это язык программирования, изначально разработанный для применения на веб-страницах, чтобы сделать их динамическими и "живыми". Программы на языке JavaScript выполняются и распространяются в текстовом виде и называются - "скриптами". Скриптам не нужна компиляция или специальная подготовка для выполнения.

В настоящее время JavaScript может выполняться на клиентской части веб-приложений - в браузере, на серверной части или на любом другом устройстве, с помощью специальных программ, называющующихся «движком». Браузеры имеют собственный "движок" для выполнения скриптов. V8 – в Chrome, Opera и Edge, SpiderMonkey – в Firefox и т.д.

JavaScript не предоставляет низкоуровнего доступа к памяти и является "безопасным" языком программирования. Возможности JavaScript зависят от окружения, в котором он выполняется. При выполнении в браузере - доступно всё, что связано с веб-страницами и веб-сервером, с некоторыми ограничениями, обеспечивающими безопасность пользователей веб-страниц.
